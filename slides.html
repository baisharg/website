Resultado: Fragmentos de código aislados, el contexto se pierde, apps chiquitas

🧠 El Problema Posta: Context Window

📉 El modelo pierde precisión con mucho contexto

(LoCoDiff Benchmark - Claude Opus 4.1)
• 2-21K tokens → ~95% la pega
• 21-35K tokens → ~80% anda bien
• 36-60K tokens → ~50% comienza a fallar
• 60-98K tokens → ~20% cualquiera

🎯 Se pierde lo importante

El modelo no encuentra los detalles que importan cuando hay mucho texto

♻️ Un solo chat = El sistema se deteriora completamente

Cada cambio agrega basura hasta que explota todo
